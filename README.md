# 🩸 LifeX – Blood Donation Mobile App

LifeX is a Flutter-based mobile application designed to simplify blood donation and help users quickly connect with blood donors and nearby blood donation camps. The application provides donor registration, blood search, donation eligibility assessment, AI-powered assistance, and location-based camp discovery, making the blood donation process more accessible and efficient.

---

## ✨ Features

- 👤 User Registration & Login
- 🩸 Become a Blood Donor
- 🔍 Search Blood Donors by Blood Group & City
- 📍 Nearby Blood Donation Camps using Google Maps
- 🤖 AI Blood Donation Chatbot
- ✅ Blood Donation Eligibility Quiz
- 👤 User Profile Management
- 📱 Clean and User-Friendly Interface

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/Kaviyasree27/LifeX-Blood-Donation-Mobile-App.git
```

2. Navigate to the project directory

```bash
cd LifeX-Blood-Donation-Mobile-App
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
